# Clone de la page d'accueil Google

 **[Voir le site en direct ici](https://litissia-gmz.github.io/Clone-Google-Homepage/)**

##  Objectifs techniques
- **Structure sémantique :** Utilisation pertinente des balises HTML5 (`<header>`, `<main>`, `<nav>`, `<footer>`).
- **Mise en page CSS :** Utilisation avancée de **Flexbox** pour le centrage et l'alignement des éléments (barre de recherche, boutons, footer).
- **Design UI :** Intégration d'icônes (FontAwesome) et de polices externes (Google Fonts : Inter, Poppins).
- **Interactivité :** Gestion des états de survol (`:hover`) et des focus (`:focus-within`) pour une meilleure expérience utilisateur.

##  Technologies Utilisées
- HTML5
- CSS3 (Flexbox, variables, transitions)

##  Comment visualiser le projet ?
Il suffit de cloner ce dépôt et d'ouvrir le fichier `index.html` dans n'importe quel navigateur web moderne.
